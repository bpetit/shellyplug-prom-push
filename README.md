# Shelly Plug prometheus push exporter

## Get administrative access to the shelly plug

1. Hold the button for more than 10 seconds to reset it to factory settings
2. Connect to the shelly plug's own wifi network
3. Connect to http://192.168.33.1/ with your browser
4. Go to Internet & Security > Wifi Mode - Client
5. Check "Connect the Shelly device to an existing Wifi Network"
6. Fill in your SSID and wifi password and save
7. Disconnect from shelly wifi
8. Connect to your wifi network
9. Scan for Shelly plug ip address given by DHCP server
10. API is accessible over HTTP
